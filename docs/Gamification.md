# 1. Objetivo

A gamificação do FranciOS existe para incentivar hábitos saudáveis, evolução pessoal e consistência.

O objetivo NÃO é prender o usuário ao aplicativo.

O objetivo é ajudá-lo a viver melhor.

---

# 2. Filosofia

Toda recompensa deve representar evolução real.

Nunca tempo de tela.

Nunca quantidade de cliques.

Nunca ações sem significado.
